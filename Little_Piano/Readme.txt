Title: Little_Piano.zip

Description: This  applicatiom makes use of API known as BeepAPI.
There is a textbox behind The Frame of the Form which always has 
the focus whenever the key is pressed.
Make sure to make Tabstop property to False for the two Scrollbar.
The Piano will function only during the keypress provided
there is focus to the Form.
The keypress (a to z) and spaceBar is only accepted to make sound.
